# Plan: AI Tutor Lớp 1 - Người bạn học tập thông minh
Created: 2026-04-23 12:20
Status: 🟡 In Progress

## Overview
Dự án xây dựng một ứng dụng Web giúp học sinh lớp 1 tự kiểm tra bài tập Toán/Tiếng Việt qua ảnh chụp. AI sẽ nhận diện lỗi sai, đưa ra gợi ý bằng giọng nói và tặng Sticker khen thưởng để khuyến khích bé tự sửa bài.

## Tech Stack
- Frontend: Next.js + TailwindCSS (Giao diện đẹp, mượt)
- Backend: Next.js API Routes (Xử lý logic đơn giản)
- AI: Google Gemini Vision API (Nhận diện hình ảnh và chữ viết tay)
- Voice: Web Speech API hoặc Edge TTS (Phát giọng nói hướng dẫn)
- Database: Supabase hoặc Local Storage (Lưu trữ tiến độ và bộ sưu tập Sticker của bé)
